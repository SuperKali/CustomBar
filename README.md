# Custom Bar Plugin for GenisysPro


## how does it work?

* This plugin allows you to have a customizable HUD on your server!
* You can add options to show players in the game!
* In the future new updates!

## Format HUD List

* {tps} 
* {load}
* {player}
* {max_players}
* {line}
* {coins} EconomyAPI necessary!
* & and §
